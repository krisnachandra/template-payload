## Quick Start

- Use minimum `node 18.18`
- Copy `.env.example` to `.env`. Setup the vars
- Open terminal

  ```

  yarn

  # preview dev
  yarn dev

  # build production
  # but still buggy caused by plaiceholder & MJML
  ```
